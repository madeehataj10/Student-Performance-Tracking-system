# Student Performance Tracking System

##  Overview
The Student Performance Tracking System is a Python-based application designed to manage and analyze student academic performance. It allows users to store student details, record subject-wise marks, calculate performance metrics, and generate insights to help evaluate student progress.

This system automates manual record-keeping and provides a structured approach to monitoring academic results.

---

## Features
- Add and manage student details
- Store and update subject-wise marks
- Calculate total marks, percentage, and performance metrics
- Track academic progress of students
- Data storage using a lightweight database
- Simple and user-friendly implementation

---

## Technologies Used
- **Programming Language:** Python
- **Database:** SQLite
- **Libraries:** sqlite3, Pandas (optional for analysis)
- **Tools:** VS Code / Jupyter Notebook
